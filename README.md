
# 🔐 Caesar Cipher Program (Python)

A simple Python program that implements the **Caesar Cipher algorithm** to encrypt and decrypt messages.  
This project is menu-driven and allows users to repeatedly choose between encryption, decryption, or exiting the program.

---

## 📌 Features

- 🔒 Encrypt messages using Caesar Cipher
- 🔓 Decrypt messages back to original text
- 🔁 Menu-driven interface (Encrypt / Decrypt / Quit)
- 🔤 Supports uppercase and lowercase letters
- ␣ Preserves spaces and special characters
- 🔢 User-defined shift value (1–25)

---

## 🛠️ How It Works

The Caesar Cipher shifts each letter in the message by a fixed number (shift value):

- **Encryption:** Moves letters forward
- **Decryption:** Moves letters backward
